# base-page

简单点，就弄个可以修改后自动刷新的简单页面。
1.自动刷新工具[browsersync](http://browsersync.cn/)
	
		npm install --save-dev browser-sync

2.less实时转css[koala](http://koala-app.com/index-zh.html)


3.浪起来(自己看看文件放在啥位置)
	
		browser-sync start --server --files "css/*.css, *.html"
